校园出入口客流预测疏导平台使用指南

一、平台简介

本平台为校园师生提供南北校门客流量查询与出行参考服务，系统结合实际调研数据，推算历史及未来一周客流变化，帮助大家避开拥堵路段，提升日常通行效率。

二、首页功能使用

1. 页面顶部自动显示系统实时日期、星期与时间，数据实时刷新更新。

2. 日期栏可切换查看昨日、今日以及未来七天的客流情况，点击对应日期即可切换查看。

3. 页面中部会实时判定校门通行状态，分为畅通、轻微缓行、严重拥堵三种提示。

4. 当日时间会自动将当前时段卡片居中展示，左右滑动卡片可查看全天24小时人流分布。

5. 系统根据客流数据自动生成出行建议，推荐合适通行校门，并预估拥堵持续时长。

6. 下方支持切换曲线图、柱状图、扇形图，直观查看客流变化规律与南北门人流对比。

三、数据查询功能操作

1. 点击导航栏数据查询，选取想要查看的日期。

2. 点击查询按钮，即可加载当日全天客流变化曲线图表。

3. 录入真实数据后自动生成预测数据，首页、查询页数据实时同步。

4. 暂无录入数据时无法展示客流信息。

5. 如需留存数据，点击导出Excel按钮，可将当日客流表格保存至本地设备。

四、日常使用小提示

1. 早晚上下课时段北门人流量普遍偏高，容易出现排队拥堵，可参考平台建议错峰出行。

2. 平台预测数据依托真实录入数据生成，参考性更强。

3. 普通用户仅可查看、查询与导出数据，无法修改平台内部客流信息。

4. 滑动类卡片均可鼠标左右拖拽滑动，方便浏览全部时段信息。
